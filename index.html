Project 2

Project Information

Learning Objectives

  • Gain experience using git via your CLI and Visual Studio Code (VSCode) Source Control
  • Gain experience writing and executing non-web server Node.js JavaScript code
  • Practice refactoring JavaScript code

Summary

This project focuses on developing skills in using Git via both the command line and Visual Studio Code (VSCode), while practicing JavaScript code refactoring and function expression techniques in a Node.js environment. The project starts by setting up a Git repository, refactoring a provided JavaScript file by isolating code into functions, removing global variables, and improving code readability. Finally, commit new functions incrementally using Git, ensuring each change runs correctly in VSCode.
